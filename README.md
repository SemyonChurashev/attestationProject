# Проект с практическими заданиями по плану обучению автоматизации на Python 

## Структура проекта 
```
project/
│
├── basePythonModule # Содержит практические задачи по блоку 'Программирование на Python'
│
├── dateTimeModule # Содержит практические задачи по блоку 'Работа с датами, Модуль datetime
│
├── jsonModule # Содержит практические задачи по блоку 'Работа с JSON'
│
├── oopModule # Содержит практические задачи по блоку 'ООП -  основные понятия'
│
├── pageObjectModule # Содержит практические задачи по блоку 'Паттерн автоматизации Page Object'
│
├── pytestModule # Содержит практические задачи по блоку 'Работа с Pytest'
│
├── requestLibModule # Содержит практические задачи по блоку 'Запросы. Библиотека requests'
│
├── seleniumModule # Содержит практические задачи по двум блокам 'Selenium + Локаторы, Поиск по XPath, CSS.'
│
├── tmp # Каталог для хранения временных файлов json + txt
│
├── utils # содержит утилиту для опредления ROOT_DIR
│
└── requirements.txt # Зависимости проекта
````
